Button

CSS Buttons are used to decorate the web pages by applying the various styling properties to the button Buttons are used for event processing and interacting with the user.

Example of Primary Buttons

Primary button gives more semantic meaning to the element we are applying.Provides extra visual weight and identifies the primary action in a set of buttons.

Example of Secondary Buttons

Secondary buttons are the alternative we give users to the primary action. Secondary buttons are used to perform non-primary actions (e.g. Cancel, Reset Filters, etc.)

Example of Icon Buttons with text

An icon button with text is an icon that triggers action written beside it

Floating Action Button

A floating action button performs the primary, or most common, actions on the screen.They are usually in circular shape with an icon at center.They are of three types regular, mini and extended.
